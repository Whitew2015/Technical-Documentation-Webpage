Introduction

Python is an interpreted, object-oriented, high level programming language that can:

  • Be used on a server to create web applications
  • Be used alongside software to create workflows
  • Connect to database systems and read and modify files
  • Be used to handle big data and perform complex mathematics
  • Be used for rapid prototyping or production-ready software development

There are several reasons why people prefer using Python to other programming languages:

  • Python works on different platforms such as Windows, Mac and Linux
  • Python has a simple syntax similar to the English language
  • The syntax allows developers to write programs with fewer lines than some other programming languages
  • Python runs on an interpreter system, meaning that code can be executed as soon as it is written
Get Started

Steps in getting started:

  1. For Windows PC, check if you have Python installed by searching it in the start bar or run the following Command Line(cmd.exe): 
                        
                        C:\Users\Your Name>python -- version
  2. For Linux or Mac, open the command line(Linux) or Terminal(Mac) and type: 
                        
                        python --version
  3. If you don't have Python installed on your computer you can download it for free at Python.org
  4. Once it is installed, the way to run a python file is like this on the command line: 
                        
                        C:\Users\Your Name>python filename.py
Syntax

Python syntax can be executed by writing directly in the Command Line: 

                
                    >>> print("Hello, World!")
                    Hello, World!

Or creating a python file on the server, using the .py file extension and running it in the Command Line(see previous section).

Like other programming languages, indentation is very in Python. Python uses identation to indicate a block of code: 

                
                    if 8 > 4:
                        print("Eight is greater than four")

Python will give you a syntax error if you skip the identation: 

                
                    if 8 > 4:
                    print("Eight is greater than four")
